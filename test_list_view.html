🔍 Cách Kiểm Tra

  1. Truy cập: http://open-crm.ddev.site/admin/crm/teams
  2. Đăng nhập bằng tài khoản admin
  3. Tìm phần "User Team Assignments" ở bên phải
  4. Nhấn nút "List" (bên cạnh nút "Cards")
  5. ✅ Bạn sẽ thấy table hiển thị với các cột:
    • User (với avatar)
    • Email
    • Current Team
    • Roles
    • Action (dropdown + Save button)
  6. Nhấn nút "Cards" để chuyển về card view

🎯 Tính Năng List View

  • ✅ Hiển thị users dạng table compact
  • ✅ Avatar với initials
  • ✅ Team badges với icons
  • ✅ Role badges
  • ✅ Dropdown để chọn team
  • ✅ Save button để lưu assignment
  • ✅ Hover effect trên rows
  • ✅ Search và filter vẫn hoạt động

⚙️ JavaScript Logic

Function switchView(view) hoạt động như sau:

function switchView(view) {
  const grid = document.getElementById('users-grid');
  const tableContainer = document.getElementById('users-table-container');
  const buttons = document.querySelectorAll('.view-toggle-btn');
  
  // Update active button
  buttons.forEach(btn => {
    btn.classList.remove('active');
    if ((view === 'cards' && btn.textContent.includes('Cards')) || 
        (view === 'list' && btn.textContent.includes('List'))) {
      btn.classList.add('active');
    }
  });
  
  // Toggle views
  if (view === 'cards') {
    grid.style.display = 'grid';
    tableContainer.style.display = 'none';
  } else {
    grid.style.display = 'none';
    tableContainer.style.display = 'block';
  }
}
