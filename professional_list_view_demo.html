🎨 Chi Tiết Styling

1. Avatar với Gradient

  • .user-avatar-table - 44x44px, border-radius 12px
  • 6 gradients khác nhau dựa trên user ID
  • Box shadow cho depth effect
  • Pseudo-element ::after cho border gradient

2. Icon Boxes

  • .email-icon - 32x32px với gradient xanh dương
  • Rounded corners 8px
  • Centered icon với lucide icons

3. Team & Role Badges

  • Padding 8px 14px, border-radius 8px
  • 2px solid borders với màu matching
  • Gradient backgrounds
  • Icons với proper sizing

4. Row Hover Effects

  • Gradient background: linear-gradient(135deg, #faf5ff, #f5f3ff)
  • Left border 3px với gradient tím-xanh
  • transform: translateX(2px) - slide effect
  • Box shadow cho depth

5. Action Buttons

  • Dropdown: Custom arrow icon, hover states
  • Save button: Gradient xanh lá, shadows, lift effect
  • Smooth transitions 0.2s cho mọi interactions
